COMO FUNCIONA O CÓDIGO: esse código é uma loja online simples de pregos feito com HTML, CSS e javascript
puro.
o HTML divide o site em duas partes
< LOGIN
usúario digita o nome
clica em entrar

APLICAÇÃO:
< lista de pregos(produtos)
< campo de busca
< carrinho
< pagar com pix ou cartão 

ESTILO(CSS)
< fundo escuro 
< cards brancos para produtos
< botões simples e organizados
< Layout em grid responsivo

LOGIN: 
< salva o nome do usúario no site 
< se você já estiver com o nome ele entra direto 
< senão ele mostra pra você fazer o login 

LISTA FIXA DE PRODUTOS

ID
NOME

CARRINHO:

< se já existe aumenta o produto 
<  mas se não ele adiciona um novo 

DIMINUIR:
< diminui quantidade ou remove ele

SALVAR:
< mantém o carrinho salvo mesmo recarregando ele 

ATUALIZA CARRINHO:
< lista os itens a cima 
< calcula o total
< calcula o frete
< aplica a promoção
